# Industrial Defect Detection using CNN

## Overview

This project uses a Convolutional Neural Network (CNN) to detect industrial defects from images.

## Features

- Image preprocessing
- CNN model
- Model training
- Validation
- Prediction on new images
- Accuracy and loss visualization

## Dataset

The dataset contains images of defective and non-defective products.

Classes:

- Defective
- Non-defective

## Technologies

- Python
- TensorFlow
- Keras
- NumPy
- OpenCV
- Matplotlib

## Results

Training Accuracy: 98%

Validation Accuracy: 95%
